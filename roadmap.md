# Next feature plans

## Near-term

* scrape Kivimäe train stop times and add a new route pair Kivimäe-Laagri and Laagri-Kivimäe
* auto-detect user location, pick the closest station in route pair as the starting place for the route (e.g Tallinn-Laagri route pair: closer to Tallinn, pick Tallinn > Laagri; closer to Laagri, pick Laagri > Tallinn;)
* remember if user picked last Kivimäe-Laagri or Tallinn-Laagri route pair, and store this pair as the default
* UX/UI upgrade - make the visual more readable and professional looking
* make the route selection more intuitive and easier to change (Laagri-Tallinn vs Tallinn-Laagri) without scrolling on page

## Long-term

* add more stops for route options, all from Elron route
* user can pick their default routes that is their Home Route, e.g Laagri-Kivimäe or Laagri-Tallinn
    * their Home Route choice is persisted in a cookie and always shown as default
